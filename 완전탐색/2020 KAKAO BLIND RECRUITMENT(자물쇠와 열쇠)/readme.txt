## 문제 유형

- 완전탐색

## 문제 접근

- 답
    - 열쇠가 자물쇠의 빈곳을 모두 채우면 true, 아니면 false 반환
- 주어지는 정보
    - 열쇠 크기 m(m ≤ 20)
    - 자물쇠 크기 n (n ≤ 20)
    - 홈 0, 돌기 1
- 후보
    - 완전탐색
        - 적합
            - 추정 연산 횟수 : (20*20*4)*(20*20 + 20*20) = 1,280,000
            - 열쇠 4회전 * (자물쇠의 모든 좌표와 매칭 + 자물쇠의 빈 곳이 모두 없어지는 지 확인)
            - ⇒ 안정적인 연산 횟수로, 완전탐색 적합
- 설명
    - 열쇠 회전
        - key의 좌표를 읽는 시작 위치를 다르게 하면 90*n도 회전이 가능함
        - 예시(90도 회전)
            - y=0, x=0부터 읽는게 아닌, x=0, y=n-1부터 읽는 것
