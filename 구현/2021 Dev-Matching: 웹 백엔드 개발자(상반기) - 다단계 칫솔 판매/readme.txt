## 문제 유형

- 구현

## 문제 접근

- 답
    - 각 인원이 가지는 돈
- 주어지는 정보
    - 인원 100,000
    - 자식, 부모 관계
    - 소득이 발생하는 대상
- 후보
    - 완전탐색
        - 부적합
            - 분기점에 따른 경우의 수가 나눠지지 않기 때문에 부적합
    - 구현
        - 적합
            - n*n = 1억 연산
            - (전체 인원) * (각 인원으로 인해 발생하는 최대 연산량)
            - ⇒ 문제에서 주어지는 설명을 그대로 따라해서 코드를 구성해도 연산량 초과X
