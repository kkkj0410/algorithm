## 문제 유형

- 완전 탐색, BFS

## 문제 접근

- 답
    - 5번의 MOVE로 얻을 수 있는 최대 블록 값
- 주어지는 정보
    - 보드의 최대 크기 20*20
    - 시간 제한 1초
    - 최대 5번 MOVE
        - 완전 탐색이 가능함(제한 크기에 비해 시간 제한이 많이 널널함)
            - BFS, DFS 완전 탐색 둘 다 가능한 알고리즘일 것으로 예상됨.
            - BFS : 최대 5번에서 큐 삽입 종료
            - DFS : 최대 5번에서 재귀 호출 종료(최소 MOVE를 구하는 문제가 아니기 때문에 DFS도 가능)
            - 예상되는 연산 횟수 : 보드 크기 * 시행 횟수
                - (20*20) * (4^5) = 409600(1초 미만이니 통과 가능)
