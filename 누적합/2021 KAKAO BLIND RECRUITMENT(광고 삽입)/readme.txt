## 문제 유형

- 누적합

## 문제 접근

- 답
    - 제일 많은 시청자가 보기 위한 광고 시작 타임
- 주어지는 정보
    - 영상 최대 길이 99:59:59
    - 시청자 수 최대 300,000
- 후보
    - 완전탐색
        - 가능
            - 누적합을 써서 각 시간별 시청자 수를 배열로 기록
            - 해당 배열을 전부 훑으면서 시청자 수가 제일 많을 때를 찾아냄
- 설명
    - 누적합
        - int[] temp : 시청자의 시청 시작, 종료 시간을 표기
        - int[] logsTime : 각 시간별 시청자의 총합을 표기
