## 문제 유형

- DFS

## 문제 접근

- 답
    - 4칸으로 가질 수 있는 최대 값(대각선 불가)
- 주어지는 정보
    - 맵 크기 N*M = 250000
    - 탐색 범위 5*4 = 20
        - 테트로미노 종류 * 회전(4방향) = 20
    
    ⇒ 250000*20 = 5,000,000이므로 완전탐색 가능
    
- DFS vs BFS
    - BFS
        - BFS는 ㅗ 모양을 만들기 적절치 않음(visit 기록할 때, 타이밍이 안맞음)
    - DFS
        - 테트로미노 모양의 탐색에 있어 전부 대응 가능
    - ⇒ DFS
