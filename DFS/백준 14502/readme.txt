## 문제 유형

- DFS + BFS

## 문제 접근

- 답
    - 안전 영역의 최대 크기
- 주어지는 정보
    - 지도 크기 N*M = 64
    - 벽 3개 배치
        
        ⇒ 완전 탐색 가능
        
        - (64 C 3) * 64 = **2,666,496**
        - 64개 중에서 3개의 벽 배치 * 바이러스 퍼뜨리기
    - BFS vs DFS
        - 둘 다 사용 가능
        - (최솟값을 구하는 문제면 BFS가 유력하지만, 최대값을 구하기 때문에 둘 다 가능할 것으로 보임)
