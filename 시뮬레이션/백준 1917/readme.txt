## 문제 유형

- 시뮬레이션

## 문제 접근

- 답
    - 정육면체 주사위를 만들 수 있는 지 검사
- 주어지는 정보
    - 주사위 3개 정보
- 후보
    - 완전탐색
        - 가능할 것이라 생각됨
        - 주사위의 정보는 3개이므로, 그에 따른 모든 경우의 수 탐색으로도 시간복잡도가 크게 늘어나지는 않음
        - 하지만 방법을 찾지 못함
    - 시뮬레이션
        - 가능
            - 주사위를 굴린다고 생각하고 시뮬레이션을 구현가능
- 풀이
    - 목적
        - 주어진 주사위를 정육면체 주사위로 만들어보자.
    
    ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/9a0c0d89-33bc-42a9-96fd-4b51064a7a84/d1d22e0d-5404-4832-8b4d-b79e7f25af89/image.png)
    
    - 방법
        - 입력 주사위가 위 사진 모양 틀에 들어갈 수 있는지를 확인한다.
        - 해당 틀을 왼, 오, 위, 아래 방향으로 굴려가면서 입력값이 1일때 해당 틀 위치를 true로 채운다.
        - 주사위의 빈 곳이 없다면 입력 주사위는 정육면체 모양을 만들 수 있다.
